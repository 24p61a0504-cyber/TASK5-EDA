# Titanic EDA Project

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset.

## Files Included
- EDA_Titanic.ipynb : Main notebook with code + visualizations
- titanic.csv : Dataset used
- report.pdf : PDF version of the notebook

## Steps Performed
- Data loading & cleaning
- Univariate, bivariate, and multivariate analysis
- Correlation heatmap
- Pairplot
- Missing value analysis
- Summary of insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
